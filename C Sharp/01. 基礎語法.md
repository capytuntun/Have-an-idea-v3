## 字串的用法
>[!note]
>常用的 C# 字串操作
- 字串的換行
``` C charp
System.Console.WriteLine("Hello \nMr.White");
```
- 跳脫字元
``` C sharp
System.Console.WriteLine("Hello \"Mr.White");
```
- 字串相加
``` C sharp
System.Console.WriteLine("Hello " + "Mr.White");
```
- 字串相加(透過變數)
``` C sharp
string phrase = "Hello ";
System.Console.WriteLine(phrase + "Mr.White");
```
- 字串長度計算
``` C sharp
string phrase = "Hello Mr.White";
System.Console.WriteLine(phrase.Length);
```
- 把字串轉換為大寫
``` C sharp
string phrase = "Hello Mr.White";
System.Console.WriteLine(phrase.ToUpper());
```
- 把字串轉換為小寫
``` C sharp
string phrase = "Hello Mr.White";
System.Console.WriteLine(phrase.ToLower());
```
- 判斷子字串是否存在
``` C sharp
string phrase = "Hello Mr.White";
System.Console.WriteLine(phrase.Contains("Mr"));
```
- 字串內特定位置的值
``` C sharp
string phrase = "Hello Mr.White";
System.Console.WriteLine(phrase[1]);
```
- 字串內特定值的位置
``` C sharp
string phrase = "Hello Mr.White";
System.Console.WriteLine(phrase.IndexOf("H"));
```
- Substring用法
``` C sharp
string phrase = "Hello Mr.White";
System.Console.WriteLine(phrase.Substring(2));
```
``` C sharp
string phrase = "Hello Mr.White";
System.Console.WriteLine(phrase.Substring(6, 3));
```
## 數字用法
>[!note]
>整數與浮點樹的用法
-  加減乘除
``` C sharp
System.Console.WriteLine(5 + 2);
System.Console.WriteLine(5 - 2);
System.Console.WriteLine(5 * 2);
System.Console.WriteLine(5 / 2);
```
- 浮點數運算 : 只要有一個是浮點數，輸出就是浮點數
``` C sharp
System.Console.WriteLine(5 / 2.0);
```
- 絕對值
``` C sharp
System.Console.WriteLine(System.Math.Abs(-10));
```
- 次方運算
``` C sharp
System.Console.WriteLine(System.Math.Pow(2,3));
```
- 開跟號
``` C sharp
System.Console.WriteLine(System.Math.Sqrt(64));
```
- 找最大的數
``` C sharp
System.Console.WriteLine(System.Math.Max(64, 101));
```
- 找最小的數
``` C sharp
System.Console.WriteLine(System.Math.Min(64, 101));
```
- 四捨五入
``` C sharp
System.Console.WriteLine(System.Math.Round(4.5));
```

# if 判斷句
- If 條件式
``` C sharp
bool hungry = true;
if (hungry)
{
    System.Console.WriteLine("我就去吃飯");
}
```
- If-else 條件式
``` C sharp
bool rainy = false;
if (rainy)
{
    System.Console.WriteLine("我就開車去上班");
}
else
{
    System.Console.WriteLine("我就走路去上班");
}
```
- if-else if-else 條件式
``` C sharp
int score = 60;

if (score == 100)
{
    System.Console.WriteLine("我給你1000元");
}
else if (score >= 80)
{
    System.Console.WriteLine("我給你500元");
}
else if (score >= 60)
{
    System.Console.WriteLine("我給你100元");
}
else
{
    System.Console.WriteLine("你給我300元");
}
```
- 兩個條件式
``` C sharp
int score = 100;
bool rainy = true;
if (score == 100 && rainy)
{
    System.Console.WriteLine("我給你1000元");
}
else
{
    System.Console.WriteLine("你給我100元");
}
```
# 迴圈
>[!note]
>While迴圈、For-Loop迴圈

### While-Loop
- While-Loop
``` C sharp
int num = 1;
while (num <= 5)
{
    System.Console.WriteLine(num);
    num++;
}
```
- Do-While
``` C sharp
do
{
    System.Console.WriteLine(num);
    num++;
}
while (num <= 5);
```
### For-Loop
``` C sharp
for (int i = 0; i<=5; i++)
{
    System.Console.WriteLine(i);
}
```

``` C sharp
int[] nums = { 2, 45, 3, 34, 2, 1 };

for (int i = 0; i<6; i++)
{
    System.Console.WriteLine(nums[i]);
}
```
