# Class
>[!note]
>類似模板的概念
- Person.cs
``` c sharp
class Person
{
    public double height;
    public int age;
    public string name;
}
```
- Program.cs
``` c sharp
Person person1 = new Person();

person1.height = 163.5;
person1.name = "小白";
person1.age = 80;

System.Console.WriteLine(person1.height);
```

# NameSpace
>[!note]
>管理模板的方式
- Person.cs
``` c sharp
namespace Animal
{
    class Person
    {
        public double height;
        public int age;
        public string name;
    }
}
```
- Program.cs
``` c sharp
Animal.Person person1 = new Animal.Person();

person1.height = 163.5;
person1.name = "小白";
person1.age = 80;


System.Console.WriteLine(person1.height);
```
``` c sharp
using Animal;
Person person1 = new Person();

person1.height = 163.5;
person1.name = "小白";
person1.age = 80;


System.Console.WriteLine(person1.height);
```

# Method
>[!NOTE]
>等於在Namespace裡面的Function
- Person.cs
``` c sharp
using System;
namespace Animal
{
    class Person
    {
        public double height;
        public int age;
        public string name;
        public void SayHi()
        {
            Console.WriteLine("你好啊，我叫" + name);

        }
        public bool IsAduit()
        {
            if (age >= 18)
            {
                return true;
            }
            else
            {
                return false;
            }
        }
    }
}
```
- Program.cs
``` c sharp
bool IsAduit = person1.IsAduit();

Console.WriteLine(IsAduit);
```
>[!warning]
要可以輸入便數可以用以下方法
- Person.cs
``` c sharp
namespace Animal
{
    class Person
    {
        public int add(int num1, int num2)
        {
            return num1 + num2;
        }
    }
}
```
- Program.cs
``` c sharp
int add = person1.add(10, 15);
System.Console.WriteLine(add);
```
# constructor 建構方法
>[!note]
>執行一個Class的時候會自動執行

``` c sharp
namespace Animal
{
    class Person
    {
        public double height;
        public int age;
        public string name;

        public Person()
        {
            Console.WriteLine("創建成功");
        }
    }
}
```

也可以傳字串進去

``` c sharp
class Person
{
    public double height;
    public int age;
    public string name;

    public Person(string qq)
    {
        Console.WriteLine("創建成功" + qq);
    }
}
```

``` c sharp
Person person = new Person("爽拉");
```
